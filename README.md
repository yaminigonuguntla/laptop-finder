# LaptopFinder

Filter laptops by specs, search by name, and compare up to 4 side by side.

HTML, CSS, and JS. Data lives in `laptops.csv`.

## Run it

```bash
python -m http.server 5500
```

Open http://localhost:5500

Filters use include/exclude checkboxes — hit **Apply filters** when you're ready. There's a compare bar at the bottom and a price popup on each card (made-up prices; store links are real searches).
